# 🧰 Curso Design Patterns
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-red)
> Este repositório não contém código desenvolvido nas aulas, mas sim soluções independentes focadas na lógica de negócio e na resolução de problemas práticos.

Toda a construção deste repositório está interligada aos conhecimentos adquiridos com [@RenatoAugustoFS](https://github.com/RenatoAugustoFS), [@luizomf](https://github.com/luizomf) e do livro *Design Patterns com Java: Projeto orientado a objetos guiado por padrões* ([casa do código](https://www.casadocodigo.com.br/products/livro-design-patterns)).
