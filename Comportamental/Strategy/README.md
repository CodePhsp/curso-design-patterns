# Strategy

Define uma família de algoritmos, encapsula cada um deles e os torna intercambiáveis.
Usa composição para permitir a troca de algoritmos em tempo de execução.

## Aplicação

- Quando há excesso de condicionais
- Necessidade de isolar a regra de negócio do algoritmo
- Variantes de um mesmo algoritmo

## Relação com SOLID

- SRP - Single Responsability Principle
- OCP - Open/Closed Principle
- DIP - Dependency Inversion Principle

## Variações do Strategy

- Não há vairiações.
